# 🧾 Facturador-telcel - Extractor de Datos de Facturas Telcel

Aplicación de escritorio para extraer y procesar automáticamente datos de facturas PDF del CAD Telcel, agilizando el flujo de trabajo administrativo.

## ✨ ¿Qué hace?

Extrae automáticamente de facturas PDF los siguientes datos de equipos celulares:

- 📋 Nombres y descripciones de equipos
- 🔢 Cantidades
- 💰 Precios e IVA
- 📧 Correos electrónicos
- 📶 ICCID (para SIMkit y Express)

## 🛠️ Tecnologías usadas

- Python 3
- Tkinter – interfaz gráfica
- PyPDF2 / pdfplumber – extracción de datos desde PDF

## ▶️ Cómo ejecutarlo

1. Clona el repositorio:
````bash
   git clone https://github.com/Alherian1/facturador.git
   cd facturador
````

2. Instala las dependencias:
````bash
   pip install pdfplumber
````

3. Ejecuta la aplicación:
````bash
   python Facturador.py
````

> Asegúrate de tener el archivo `logo.ico` en la misma carpeta que el script.

## 📌 Uso

1. Abre la aplicación
2. Carga la factura en formato PDF
3. El programa extrae automáticamente todos los datos relevantes
4. Exporta o visualiza la información procesada

## 💡 Lo que aprendí

- Extracción de datos desde archivos PDF con Python
- Automatización de procesos administrativos reales

- Manejo de datos estructurados y no estructurados
